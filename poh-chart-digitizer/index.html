POH Chart Digitizer

Controls: Scroll to pan • Ctrl+Scroll to zoom • Hold Z for magnifier
Crosshair: Enable to align rotation with chart grid lines
Workflow: Load image → adjust rotation → calibrate Y then X axes → trace lines → compute

1. Load Chart Image

2. Calibrate Distance Axis (Y)

Click one point per distance value on any horizontal grid line. This Y calibration is shared across all sections.

3. Calibrate Gross Weight Section (X only)

Click X points on the weight axis.

4. Calibrate Outside Air Temperature Section (X only)

Click X points on the temperature axis.

5. Calibrate Wind Section (X only) optional

Skip if your chart has no wind section. Negative = tailwind, positive = headwind.

6. Calibrate Obstacle Section (X only) optional

Skip if your chart has no obstacle height section.

7. Trace Altitude Lines — Outside Air Temperature

Select an altitude, click along its line from left to right.

8. Trace Reference Curves — Gross Weight

Trace the guide curves on the gross weight section.

9. Trace Reference Curves — Wind optional

Skip if not calibrated. Trace the guide curves on the wind section.

10. Trace Reference Curves — Obstacle optional

Skip if not calibrated. Trace the guide curves on the obstacle height section.

11. Compute & Export

For each temp+altitude, crosses through each section and reads distance.
